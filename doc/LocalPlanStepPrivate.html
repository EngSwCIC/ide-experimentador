    Validates drawer parameters for a given record. Expects the presence of an “action” key in the parameter hash, with a specified size. Additionally, checks if the action command is either “open” or “close”.

    # File app/models/local_plan_step_private.rb, line 20
def validate_drawer(record)
  validate_parameter(record, ["action"], [1])
  valid_actions = %w(open close)
  record.errors.add(:base, "Incorrect action command") unless valid_actions.include?(record.parameter["action"])
end

    Generic method for validating parameters based on provided keys and sizes. Adds errors to the record if the parameter formatting or characteristics are incorrect.

    # File app/models/local_plan_step_private.rb, line 28
def validate_parameter(record, keys, sizes)
  keys.each_with_index do |key, index|
    unless record.parameter.has_key?(key) && record.parameter.size == sizes[index]
      record.errors.add :base, "Incorrect parameter formatting or characteristics"
      break
    end
  end
end
